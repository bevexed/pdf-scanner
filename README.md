# 账单截图导出系统

一个本地桌面工具:导入快递账单 PDF,按运单号查询,导出已打码(遮盖寄件人与金额)的账单截图。全程离线,仅连本机 127.0.0.1。

## 启动

### 交付版(推荐,给财务/客户)

双击程序图标即可打开**独立窗口**,无需终端、无需浏览器:

- **Windows**:双击 `账单截图导出系统.exe`
- **macOS**:双击 `账单截图导出系统.app`

首次在 Windows 运行若被 Defender 拦,右键「仍要运行」。

### 开发版(命令行)

```bash
cd /Users/b/web/pdf-scanner
venv/bin/python desktop.py     # 桌面窗口模式(交付形态)
# 或
venv/bin/python app.py         # 纯 Flask 调试模式,手动访问 http://127.0.0.1:5000
```

## 使用步骤

1. **导入 PDF**:选择账单 PDF,点导入,等「已入库 N 票」。可反复导入累积(按内容自动去重)。
2. **查询单号**:选「杂费/关税」,粘贴运单号(每行一个),点查询。
   同一单号在多期账单出现时,会分别列出(显示来源账单号)。
3. **导出**:「有杂费/有关税」的可预览、单张或批量导出 ZIP。寄件人与金额已实心遮盖。
   若显示「打码不完整」,说明该票字段未识别全,不会导出,请人工处理。

## 窗口 / 退出行为

- **Windows**:关闭窗口(✕)= 隐藏到右下角托盘,程序继续后台运行;托盘图标右键「显示」恢复窗口、「退出」关闭程序。导入进行中退出会二次确认。
- **macOS**:首版无菜单栏图标,关闭窗口(✕)= 退出程序;导入进行中会二次确认。

## 常见问题

- **Windows 打开后白屏**:极少数老机器缺 Edge WebView2 Runtime,双击随包 `WebView2 安装器` 装好后重开即可(Win10/11 通常已预装)。macOS 用系统自带 WebKit,无需额外安装。
- **数据存哪**:与程序同目录的 `data/` 文件夹(账单库、导出图)。中断导入不污染已入库数据,重新导入即可。

## 打包

- Windows:在 Windows 机器上运行 `build_win.bat`,产物在 `dist\`。
- macOS:运行 `bash build_mac.sh`,产物 `.app` 在 `dist/`。
